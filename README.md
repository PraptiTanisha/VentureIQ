VentureIQ AI

AI-powered startup analysis platform that predicts business success using Machine Learning.

Tech Stack

Frontend
- React
- TypeScript
- Vite
- Tailwind CSS

Backend
- Node.js
- Express.js

Machine Learning
- Python
- Scikit-learn
- Pickle (.pkl)

Features

- Business Success Prediction
- Startup Risk Analysis
- Growth Potential Score
- Investment Analysis
- Modern Dashboard
- REST API

Run Project

Frontend

```bash
cd artifacts/venture-iq
pnpm install
pnpm dev
```

Backend

```bash
cd artifacts/api-server
pnpm run build
$env:PORT=3000
node dist/index.mjs
```
